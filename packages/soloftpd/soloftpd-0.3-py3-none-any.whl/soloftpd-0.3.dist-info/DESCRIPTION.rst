========
soloftpd
========

Config
======

.. code-block:: json

   {
     "address": "127.0.0.1",
     "port": 21,
     "passive-ports": [30000, 50000],
     "masquerade-address": null,
     "username": "spam",
     "password": "egg",
     "directory": "/path/to/user/",
     "permission": "elradfmw"
   }

Command
=======

start server::

   $ soloftpd

use custom config file(default: /etc/soloftpd.conf)::

   $ soloftpd --config=/path/to/soloftpd.conf

make password hash::

   $ python -m soloftpd.authorizers your_password
